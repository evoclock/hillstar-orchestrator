Hillstar Orchestrator
=====================

**Security and reproducibility-first workflow orchestrator for research environments.**

Hillstar is a research-grade workflow orchestration system designed for
multi-agent AI pipelines and classification tasks. It provides a modular
architecture with support for multiple LLM providers, governance policies,
and comprehensive execution tracing.
